# Meu Script

## Como executar o arquivo .sh
No terminal, use o comando:

./calculadora.sh

Se precisar dar permissão antes:
chmod +x calculadora.sh

---

## O que o código em Python faz
Este script realiza operações básicas de matemática, como:
- adição
- subtração
- multiplicação
- divisão

Para executar:
python3 calculadora.py
